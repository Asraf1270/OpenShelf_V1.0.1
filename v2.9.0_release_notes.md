# 🚀 OpenShelf v2.9.0 — The "Admin Renaissance" & Intelligent Discovery

**Release Date:** May 10, 2026

A transformative update that brings the entire **Administrative Suite** into the modern era. This release harmonizes the management interface with our premium brand identity, introduces global dark mode for administrators, and implements intelligent automation for library categorization.

## 🌟 Highlights

- 🎨 **Admin Suite Renaissance** — Complete visual modernization of the entire administrative suite (Dashboard, Users, Books, Requests, Reports, Categories, Logs, Backup) with our premium "Deep Indigo & Muted Teal" design system.
- 🌗 **Global Admin Dark Mode** — Full, native dark mode support for all administrative surfaces, ensuring a consistent and comfortable experience across the entire management platform.
- 🧠 **Intelligent Category Sync** — Reimagined Category Management with an automated "Collect" engine that identifies and syncs missing categories from the book inventory in real-time.
- 📱 **Responsive Admin Architecture** — Mobile-first refactor of all admin data tables, interactive charts, and management grids, optimized for seamless control on any device.
- 🛡️ **Advanced System Monitoring** — Modernized system logs and backup management with improved terminal-style readability and intuitive action workflows.

---

## 🔧 Detailed Improvements

### 🎨 Modernized Admin UI
The legacy indigo/violet theme has been replaced with the new **Deep Indigo (#2C3E50) and Muted Teal (#4C9F8A)** palette.
- **Dashboard**: Refreshed statistics cards and interactive Chart.js growth tracking.
- **Management Grids**: Standardized 24px border-radii, premium shadows, and fluid hover states.
- **Brand Alignment**: Integrated the new simplified OpenShelf logo and wordmark across the header.

### 📚 Intelligent Categorization
- **Auto-Sync Engine**: The `loadCategories` logic now automatically detects and adds categories found in the books table that are missing from the master list.
- **Manual Collect Action**: Added a "Collect Categories" trigger in the admin UI for one-click inventory synchronization.
- **Real-Time Counting**: Updated the categorization logic to show the "real" number of books per category directly from the database.

### 📱 Responsive Refactor
- **Mobile-First Tables**: All administrative data tables (Users, Books, Requests) now support horizontal scrolling and optimized cell padding for touch devices.
- **Fluid Layouts**: The reports and analytics charts now adapt dynamically to screen size, maintaining full legibility on smartphones.

### 🛡️ System & Security
- **Terminal Aesthetics**: Reimagined the **System Logs** viewer with a high-contrast terminal theme and color-coded status levels (Error/Warning/Info).
- **Safe Backups**: Modernized the **Backup Manager** with intuitive status indicators and simplified restoration workflows.

---

## 📂 Technical Changes
- **CSS Architecture**: Migrated all admin pages to a unified CSS variable system (`:root` tokens).
- **Database Logic**: Optimized subqueries in `admin/categories.php` for faster book counting.
- **Theme Logic**: Integrated `[data-theme="dark"]` selectors across all administrative PHP files.
